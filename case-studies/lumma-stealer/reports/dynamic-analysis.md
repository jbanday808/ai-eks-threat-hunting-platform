# Dynamic Analysis Summary

## Purpose

Dynamic analysis reviewed recorded behavior from the controlled Windows lab using Noriben, Process Monitor, and Microsoft Defender telemetry.

## Findings

* Noriben recorded process, file, and network-related activity.
* Process Monitor recorded process start, thread creation, and process exit activity.
* Microsoft Defender classified the sample as malicious, blocked it, quarantined it, and recorded removal.
* The observed network indicator was documented for defensive review.

## Analyst Assessment

The recorded activity confirms that the sample executed in the isolated lab and that Microsoft Defender took remediation action. The supporting screenshots preserve the evidence without including the malware itself.
